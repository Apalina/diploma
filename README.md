# SQL RAG Assistant на базе DeepSeek-V3

Система автоматического анализа и оптимизации SQL-запросов с использованием RAG (Retrieval-Augmented Generation) и локальной LLM DeepSeek-V3.

### Требования
- Docker и Docker Compose (версия 3.8+)
- ~8 ГБ свободной оперативной памяти
- 20+ ГБ свободного места на диске
